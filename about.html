About This Web App

This web app was created to demonstrate how to build simple multi-page websites using Vite.

👨‍🏫 Purpose

It helps learners understand how web pages are connected and how navigation works in a real web app.

🔧 Features

  • Multi-page structure (e.g., Home, About, Contact)
  • Fast development with Vite
  • Easy to deploy with Netlify

📚 Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Vite
